สิ่งที่เรียนรู้

การเริ่มต้นเรียนรู้การเขียนโปรแกรม

การเรียนรู้ของผมเริ่มต้นจากความสนใจในการสร้างเว็บไซต์ ผมได้เริ่มศึกษาจากแหล่งข้อมูลออนไลน์ที่หลากหลาย โดยเฉพาะ YouTube และ W3Schools ซึ่งเป็นแหล่งเรียนรู้ที่ยอดเยี่ยมสำหรับผู้เริ่มต้น

HTML - โครงสร้างพื้นฐานของเว็บไซต์

ผมเริ่มต้นการเรียนรู้ด้วย HTML ซึ่งเป็นภาษาพื้นฐานสำหรับการสร้างโครงสร้างเว็บไซต์ ได้เรียนรู้เกี่ยวกับแท็กต่างๆ และวิธีการจัดองค์ประกอบบนหน้าเว็บ

สิ่งที่เรียนรู้เกี่ยวกับ HTML:

  • โครงสร้างพื้นฐานของเอกสาร HTML
  • การใช้แท็กต่างๆ เช่น headers, paragraphs, lists
  • การสร้างลิงก์และการเชื่อมโยงไปยังหน้าต่างๆ
  • การแทรกรูปภาพและสื่อมัลติมีเดีย
  • การสร้างตารางและฟอร์ม

CSS - การออกแบบและตกแต่งเว็บไซต์

หลังจากเรียนรู้ HTML แล้ว ผมได้ศึกษาต่อเกี่ยวกับ CSS เพื่อทำให้เว็บไซต์มีความสวยงามและน่าสนใจมากขึ้น CSS ช่วยให้สามารถปรับแต่งสี รูปแบบ และการจัดวางองค์ประกอบบนหน้าเว็บได้

สิ่งที่เรียนรู้เกี่ยวกับ CSS:

  • การเลือกองค์ประกอบด้วย selectors
  • การกำหนดสีและพื้นหลัง
  • การจัดรูปแบบข้อความและฟอนต์
  • การใช้ box model (margin, padding, border)
  • การจัดวางด้วย Flexbox และ Grid
  • การสร้าง responsive design
  • การใช้ animations และ transitions

JavaScript - เพิ่มความมีชีวิตชีวาให้กับเว็บไซต์

เมื่อได้พื้นฐาน HTML และ CSS แล้ว ผมได้เริ่มเรียนรู้ JavaScript เพื่อเพิ่มความมีชีวิตชีวาและการโต้ตอบให้กับเว็บไซต์ JavaScript เป็นภาษาโปรแกรมที่ทำให้เว็บไซต์สามารถทำงานแบบ interactive ได้

สิ่งที่เรียนรู้เกี่ยวกับ JavaScript:

  • พื้นฐานภาษา JavaScript (variables, operators, functions)
  • การจัดการกับ DOM (Document Object Model)
  • การจัดการกับ events และการโต้ตอบกับผู้ใช้
  • การใช้ conditionals และ loops
  • การทำงานกับ arrays และ objects
  • การใช้ localStorage เพื่อเก็บข้อมูล
  • การใช้ fetch API เพื่อดึงข้อมูลจาก server
